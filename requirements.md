# やりたいこと
・様々な記事をリストにまとめ、カテゴリやレーティングで整理したい。リストには概要の絵と、簡単な説明とリンクを記載したい。
・読んだ記事をストックして、いつでも見返せるようにしたい。
・記事の一覧は、わかりやすく構造化したトップページにまとめたい。
・スマホからもアクセスしたい。

# ルール
・記事に対してタグ付けを自動で行う。
・記事に対してレーティングができるようにする（星1~5）
・タグ、レーティングで検索できるようにする

# 記事のターゲット（継続的に変わる。今は意識しなくて良い）
・自動運転に関する技術記事。特に大手企業のブログなど（Waymo, Tesla, Wayve, Zoox, AutoX, Aurora, Gatik, Applied, Baidu, Pony.ai, WeRide, Motional, Nuro, Uberなど）
・自動運転に影響がある技術記事（Nvidia, Armや, 生成AI技術、シミュレーション技術、チップ、sensor, ECUなど）
